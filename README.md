# Step0Fail Blog

This is the source code for my personal blog hosted at [blog.step0fail.com](https://blog.step0fail.com).

## About

A simple, clean blog built with HTML and CSS, hosted on GitHub Pages.

## Local Development

Simply open `index.html` in your browser to preview the site locally.

## Deployment

This site is automatically deployed via GitHub Pages from the `main` branch.

## Custom Domain Setup

The site uses a custom domain (blog.step0fail.com) configured via the CNAME file.
